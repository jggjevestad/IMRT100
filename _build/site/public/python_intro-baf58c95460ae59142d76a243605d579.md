# Brukerveiledning: Introduksjon til Python for Geomatikk

Python er et kraftig, gratis og åpen kildekode-programmeringsspråk. Det er i dag industristandarden for databehandling, automatisering og romlig analyse innen geoinformatikk og landmåling.

---

## 📋 Oversikt
Hvorfor bruker geomatikere Python?
* **Automatisering**: Hvis du skal gjøre en prosess på 100 kartlag, skriver du heller et Python-skript enn å klikke manuelt i QGIS.
* **Databehandling**: GPS-er og totalstasjoner lagrer rådata i tekstfiler (f.eks. NMEA). Python er ypperlig til å lese disse, filtrere ut feilmålinger, og regne statistikk.
* **Integrasjon**: QGIS har en innebygd Python-konsoll (`PyQGIS`) og nesten alle geodata-biblioteker (som GeoPandas, GDAL og Shapely) styres med Python.

## 🔧 Komme i gang med Python

### 1. Installasjon
* **Windows og macOS**: Last ned og installer nyeste versjon fra [python.org/downloads](https://www.python.org/downloads/). Husk å huke av for **Add Python to PATH** under installasjonen på Windows.
* **Editor**: Vi anbefaler å skrive kode i **Visual Studio Code (VS Code)**. Last ned fra [code.visualstudio.com](https://code.visualstudio.com/) og installer den offisielle *Python*-utvidelsen.
* **Nødvendige biblioteker**: Åpne terminalen (i VS Code eller på PCen) og installer pakken `numpy` (brukes til matematikk og tabeller):
  ```bash
  pip install numpy
  ```

### 2. Kjøre Python-kode
Du kan kjøre Python på to måter:
1. **Interaktiv modus**: Skriv `python` or `python3` i terminalen. Du får opp en konsoll der du kan skrive linje for linje:
   ```python
   >>> 2 + 2
   4
   ```
2. **Skriptmodus**: Skriv koden din i en tekstfil som slutter på `.py` (f.eks. `analyse.py`) i VS Code, og kjør filen fra terminalen:
   ```bash
   python analyse.py
   ```

## 💻 Praktiske Geomatikk-Skript

### Skript 1: Omregning av Vinkelmål
Kopier denne koden inn i en fil kalt `vinkler.py`. Den konverterer mellom grader, radianer og gon (nygrader).
```python
from numpy import pi

def deg2rad(deg):
    return deg * (pi / 180.0)

def rad2deg(rad):
    return rad * (180.0 / pi)

def gon2deg(gon):
    return gon * (360.0 / 400.0)

def deg2gon(deg):
    return deg * (400.0 / 360.0)

# --- Test av vinkelomregning ---
vinkel_deg = 90.0
vinkel_rad = deg2rad(vinkel_deg)
vinkel_gon = deg2gon(vinkel_deg)

print("=== Vinkelomregning ===")
print(f"{vinkel_deg:.4f} grader  =  {vinkel_rad:.7f} radianer")
print(f"{vinkel_deg:.4f} grader  =  {vinkel_gon:.4f} gon")
```

### Skript 2: Gjennomsnitt og Standardavvik fra CSV
Dette skriptet leser en CSV-fil med koordinater (f.eks. eksportert fra QGIS i UTM-meter) og regner ut gjennomsnitt og standardavvik for X (Øst) og Y (Nord).
Lagre koden som `beregn_snitt.py` i samme mappe som koordinatfilen din (`data.csv`).
```python
import csv
import numpy as np

# --- Konfigurasjon ---
csv_file = 'data.csv'   # Navn på CSV-filen din
easting_col = 0         # Kolonneindeks for X/Øst (0 betyr første kolonne)
northing_col = 1        # Kolonneindeks for Y/Nord (1 betyr andre kolonne)

def analyser_koordinater(filnavn):
    ost_verdier = []
    nord_verdier = []
    
    with open(filnavn, 'r') as fil:
        leser = csv.reader(fil)
        header = next(leser)  # Hopper over første rad med kolonnenavn
        
        for rad in leser:
            try:
                # Konverterer tekstverdiene til flyttall (float)
                ost_verdier.append(float(rad[easting_col]))
                nord_verdier.append(float(rad[northing_col]))
            except ValueError:
                # Hopper over eventuelle rader med tekstfeil
                continue
                
    # Gjør om listene til NumPy-arrays for raskere beregning
    ost_array = np.array(ost_verdier)
    nord_array = np.array(nord_verdier)
    
    # Beregner statistikk
    snitt_ost = np.mean(ost_array)
    snitt_nord = np.mean(nord_array)
    std_ost = np.std(ost_array, ddof=1)  # ddof=1 gir utvalgs-standardavvik
    std_nord = np.std(nord_array, ddof=1)
    
    return snitt_ost, snitt_nord, std_ost, std_nord

# --- Kjøring ---
try:
    snitt_x, snitt_y, avvik_x, avvik_y = analyser_koordinater(csv_file)
    print("=== Posisjonsstatistikk ===")
    print(f"Målinger analysert fra: {csv_file}")
    print(f"Gjennomsnitt  -  Øst (X): {snitt_x:.3f} m   Nord (Y): {snitt_y:.3f} m")
    print(f"Standardavvik -  Øst (X): {avvik_x:.3f} m   Nord (Y): {avvik_y:.3f} m")
except FileNotFoundError:
    print(f"Feil: Fant ikke filen '{csv_file}'. Sjekk at skriptet ligger i samme mappe.")
```

## 💡 Tips & feilsøking

* **🚨 `IndentationError: expected an indented block`**: Python bruker ikke klammeparenteser `{}` for å markere blokker av kode. I stedet brukes **innrykk (tabs eller 4 mellomrom)**. Sørg for at all kode under `if`, `for`, `def` osv. har nøyaktig samme innrykk!
* **`ModuleNotFoundError: No module named 'numpy'`**: Du prøver å bruke et bibliotek som ikke er installert på maskinen din. Løsning: Gå i terminalen på PC-en og skriv `pip install numpy`.
* **`ValueError: could not convert string to float`**: Dette skjer hvis skriptet prøver å gjøre om tekst til tall (f.eks. en header-rad eller et tomt felt). Skriptet over håndterer dette med en `try-except`-blokk, men pass på at kolonneindeksene dine (`easting_col` og `northing_col`) peker på riktig kolonne i CSV-filen.
